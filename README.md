# ConnectFour
Connect Four AI. Currently uses the Minmax algorithm. 

###Installation
On Ubuntu or debian type linux

```bash
sudo apt-get install g++
```

###Running AI
Go to project root and tpye
```bash
g++ ConnectFour.cpp -o ConnectFour
```

Use ./NAME on linux.
Use NAME.exe on windows.
